# roulette-lotterys
TypeScript+ECMAScript6<br/>
前言：这个前端代码相对之前的本地可以运行的那个轮盘小项目结合了ts，做了一定新的尝试，并用php和mysql模拟了数据的请求！<br/>
这个ts的小程序运行起来也很简单：(前提是安装了node环境)<br/>
1. 打开cmd，cd到当前项目文件路径
```
cd F:\(具体的文件路径,我的文件在f盘所以以f盘为例)
```
2. 切换盘符：
```
F:
```
3. 初始化项目文件：
```
npm install
```
4. 对ts文件进行编译：
```
npm run tsc
```
